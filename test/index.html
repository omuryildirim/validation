Validation JS Examples

Welcome to Validation. You can fix any input as you wish. Here some examples how to validate.

There are two types of validation.

First, as you see a SearchObject. Search Object is used for empty based validation. If you want to check a input is empty or not you can simply define a Search Object.

Second, Validation Object. Validation Object is used for any validation rule. You can use Validation Object for email check, phone check, count check, date check etc. Basically any validation function you can think.

In addition to these validation types, Validation JS can control pressed keys. You can define an init object and you can select a set of character for your input.

  • Re-type Validation
  • Checkbox Validation
  • Basic Form Validation

For this email input, Validation JS checks both if email input is empty on button click and if email is typed correctly on keypress.

Here the JS which makes validation for email input.

$(document).ready(function() { var validate = new validator(); validate.addSearchObject("#email-submit", { "empty": { "#email-1": { parentTag: ".form-group", message: "notBeEmpty", type: "Label", showLabel: true } } }); validate.addValidationObject("#email-submit", { keyup: { "#email-1": { rule: validate.checkEmail, parentTag: ".form-group", message: "Please type email correctly!" } } }); validate.validate("click", "#email-submit"); });
For these password inputs, Validation JS checks both if both inputs have same value.

Here the JS which makes validation for passwords.

$(document).ready(function() { var validate = new validator(); validate.addValidationObject("#password-submit", { keyup: { "password-check": { ruleObject: [{ rule: validate.checkIfSame, elements: ["#password-1", "#password-2"], parentTag: ".form-group", message: "Please type same password!" }] } } }); validate.validate("click", "#password-submit"); });
For this checkbox input, Validation JS checks on submit if checkbox is not empty.

Here the JS which makes validation for checkbox.

$(document).ready(function() { var validate = new validator(); validate.addSearchObject("#checkbox-submit", { "checkbox": { "#checkbox-1": { parentTag: ".form-group" } } }); validate.validate("click", "#checkbox-submit"); });
For this basic form, Validation JS checks if inputs empty.

If you put 'main' tag to your 'validate' function, validation will check if there any empty inputs which have * character on their label. With this feature you can easily check if there is any empty input.

Here the JS which makes validation for basic form.

$(document).ready(function() { var validate = new validator(); validate.searchObj.main.html.label.parentTag = ".form-group"; validate.addSearchObject("#basic-form-submit", { "checkbox": { "#basic-checkbox-1": { parentTag: ".form-group" } } }); validate.validate("click", "#basic-form-submit", "main"); });
