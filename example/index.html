Validation.js V3 Examples

Welcome to Validation. Please take your form and prepare the easiest validation solutions in the universe.

As the world changes, validation can't wait too. With V3, validation creating becomes very easy. Now you can put validation tags to your inputs and then sit back and enjoy. There are two new way to create validations. First, put data-validation tags directly your input. Then call validation jQuery function in document ready. Second, select a form with jQuery selector and call validation jQuery function with your options as parameter.

You can simply put (data-validation="empty") tag for defining validation type. And (data-target="#emptySubmit") tag for defining submit button.
<div class="form-group col-sm-6 col-xs-12"> <label for="emptyFormName">Name</label> <input id="emptyFormName" data-validation="empty" data-target="#emptySubmit" type="text" class="form-control"></input> </div> <div class="form-group col-sm-6 col-xs-12"> <label for="emptyFormSurname">Surname</label> <input id="emptyFormSurname" data-validation="empty" data-target="#emptySubmit" type="text" class="form-control"></input> </div> <div class="form-group col-sm-6 col-xs-12"> <button id="emptySubmit" class="btn btn-danger" onclick="alert('Form received successfully!')" type="submit">Submit Empty Form!</button> </div>
$(document).ready(function() { $("body").validation(); });

  • (data-validation="date") tag for defining validation type.
  • (data-with="#compareDatesFormDateReWithLimitation") tag is mandatory for defining other input.
  • (data-limitation='{"limitEnd": 2, "limitStart": 0}') tag is an limitation object. You can add message object to limitation object. ("message": {"limitEnd": "Error"}, {"limitStart": "Error2"}})
  • (data-control="#compareDatesForSubmit") for defining submit button.
<div class="form-group col-sm-6 col-xs-12"> <label for="compareDatesFormDateWithLimitation">Select Date</label> <input id="compareDatesFormDateWithLimitation" data-validation="date" data-with="#compareDatesFormDateReWithLimitation" data-limitation='{"limitEnd": 2, "limitStart": 0}' data-control="#compareDatesForSubmit" type="date" class="form-control"></input> </div> <div class="form-group col-sm-6 col-xs-12"> <label for="compareDatesFormDateReWithLimitation">Select Same Date</label> <input id="compareDatesFormDateReWithLimitation" type="date" class="form-control"></input> </div>
